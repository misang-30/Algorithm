- 깊이 우선 탐색이라고 불린다.
- 그래프를 탐색할 때 쓰는 알고리즘이며 어떤 노드부터 시작해 인접한 노드들을 재귀적으로 방문하여 정점은 다시 방문하지 않으며 
- 각 분기마다 가능한 가장 멀리 있는 노드까지 탐색하는 알고리즘이다.

## 1. 알고리즘

``` C++ 은 아니고 수도 코드
DFS(u, adj)​
>	u.visited = true​
>	for each v ∈ adj[u]​
>		if v.visited == false
>			DFS(v, adj)
```

## 2. 구현한 코드

``` c++

#include <bits/stdc++.h>
using namespace std;

const int n = 6;
vector<int> adj[n];
int visited[n];

void DFS(int u) {
    visited[u] = 1;
    cout << u << "\n";

    for (int v : adj[u]) {
        if (visited[v] == 0) {
            DFS(v);
        }
    }

    cout << u << "로부터 시작된 함수가 종료되었습니다.\n";
    return;
}

int main() {
    // 그래프 간선 설정
    adj[1].push_back(2);
    adj[1].push_back(3);
    adj[2].push_back(4);
    adj[4].push_back(2);
    adj[2].push_back(5);

    // 노드 1부터 DFS 탐색 시작
    DFS(1);

    return 0;
}

/*

*/
```


## 3. 구현 방법
### 1). 방법 1 : 돌다리를 두들겨 보다.
- 만약 방문이 되어있다면(visited[ there ])이라면 방문을 하지 않고 방문이 안되어있다면 방문을 하는 코드입니다.
``` c++
void DFS(int here){​
	visited[here] = 1; ​
	for(int there : adj[here]){​
		if(visited[there]) continue;​
		DFS(there);​
	}​
}
```
### 2). 방법 2 : 그냥 일단 가서 보자.
- 일단 방문되어있던 안되어있던 무조건 DFS를 호출하고 해당 함수에서 만약에 방문되어있다면 return해 함수를 종료시키는 방법입니다
``` c++
void DFS(int here){​
	if(visited[here]) return;​
	visited[here] = 1;​
	for(int there : adj[here]){ ​
		DFS(there);​
	}​
}
```
