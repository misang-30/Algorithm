- 트리 순회(Tree traversal)는 트리 구조에서 각각의 노드를 정확히 한 번만, 체계적인 방법으로 방문하는 과정을 말합니다.
- 후위 순회, 전위 순회, 중위 순회로 둘 수 있다.
## 후위 순회
- 자식들 노드를 방문하고 자신의 노드를 방문하는 것을 말합니다.
``` python 
# 수도 코드
postorder( node )​
	if (node.visited == false) ​
		postorder( node->left ) ​
		postorder( node->right )​
		node.visited = true
```


## 전위 순회
- 먼저 자신의 노드를 방문하고 그 다음 노드들을 방문하는 것을 말합니다.
``` python
# 수도코드
preorder( node )​
	if (node.visited == false)​
		node.visited = true​
		preorder( node->left )​
		preorder( node->right )

```

## 중위 순회
- 중위순회(inorder traversal)는 왼쪽 노드를 먼저 방문 그다음의 자신의 노드를 방문하고 그 다음 오른쪽 노드를 방문하는 것을 말합니다.
``` python
# 수도 코드
inorder( node )​
	if (node.visited == false) ​
		inorder( node->left )​
		node.visited = true​
		inorder( node->right )

```
