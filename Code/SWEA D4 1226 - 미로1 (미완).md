## 현재 좌표에서 **상, 하, 좌, 우 4곳을 모두 살핀 뒤, 갈 수 있는(벽이 아니고 안 가본) 좌표들을 모조리 스택에 넣으시면(`append`) 됩니다.**

# 1.문자열 입력받기 , 미로 생성
maze = [list(map(int, input())) for _ in range(16)] 

# 2.변화량
dx = [0,0,-1,1]
dy = [1, -1, 0, 0]

# 3. 시작점 좌표 찾기 및 초기화 (스택, 방문처리)
# 미로에서 값이 2인 녀석이 시작점
startX = 0
startY = 0
for x in range(16) :
	for y in range(16) : 
		if maze[x][y] == 2 : 
			startX = x
			startY = y
			break
		
# 방문 체크용 2차원 리스트 (전부 False로 초기화) 
visited = [[False] * 16 for _ in range(16)]

# 스택 생성
stack = []
stack.append((startX, startY)) # 튜플 형태 (x, y)로 넣기

# 출발점 방문 표시!
